Coding Dojo Silesia Golf 2018 HOWTO

Rules

  1. Result must be printed in STDOUT
  2. Program gets 3 arguments (argv) in order: text, width (int) and height (int)
  3. Text argument is not blank
  4. Text has always one line (without newline character)
  5. Minimum width of box is 18 chars
  6. Minimum height (with bow too) of box is 14 lines
  7. Width and height are always even
  8. Each line must have newline char (\n) on the end
  9. Each line doesn't have additional blank chars on the end
