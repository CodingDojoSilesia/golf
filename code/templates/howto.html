Coding Dojo Silesia Golf 2018 HOWTO

Rules

  1. Result must be printed in STDOUT
  2. Program gets 4 positive integers from arguments (argv)
  3. Valid chars are space, newline char (\n) and █
  4. Each char (space too) have a color (more info here)
  5. Colors of char are: {{ terminal(colors) }}
  6. Each line must have newline char (\n) on the end
  7. Each line must have exactly width as other lines (right empty side must be filled with spaces)

Examples

Task are simple - write the shortest program who will be draw a CC logo in ASCII art {{ terminal(help1) }}

For ease of use, in examples char █ will be replaced as # and space will be replaced as dot (.) {{ terminal(help2) }}

First argument is a width of image {{ terminal(help3) }}

Second argument is a height of top part {{ terminal(help4) }}

Third argument is a height of middle part {{ terminal(help5) }}

Fourth argument is a height of bottom part {{ terminal(help6) }}
