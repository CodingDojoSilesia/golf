{{title}} HOWTO

Rules

  1. Result must be printed in STDOUT
  2. Program gets 1 argument (argv)
  3. Width of the output is always 60 + newline char (\n)
  4. Height of the output (number of lines) is always 45
  5. Empty space on right side of line must be filled with space characters
  6. Numbers are always odd (even numbers generate ugly ascii art)
  7. Each line must have newline char (\n) at the end
  8. The lines must not have any characters after newline char (\n)

Description

The task is simple - write the code to draw an easter egg: {{ terminal(help_fancy_egg) }}

First argument is a domain small language (DSL) to draw a pattern on the egg.

Format of the argument is: 

command1,arg1:command2,arg1,arg2,…,argN:command3,arg1,arg2:…:commandN,arg1

Commands

maskoff command is not required in the task, but is very helpful to show you how to draw on the egg. More info here.

zigzag

zigzag:CHAR,SIZE {{ terminal(help_zigzag) }}

maze

maze:CHAR,VSIZE,HSIZE {{ terminal(help_maze) }}

cross

cross:CHAR,SIZE {{ terminal(help_cross) }}

hstrip

hstrip:CHAR,HSIZE,HSIZE {{ terminal(help_hstrip) }}

vstrip

vstrip:CHAR {{ terminal(help_vstrip) }}

empty

empty:HSIZE
just draw N empty (ekhm! filled with spaces!) lines
