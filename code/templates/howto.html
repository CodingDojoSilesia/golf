{{title}} HOWTO

Rules

  1. Result must be printed in STDOUT
  2. Program gets 1 argument (argv)
  3. Width of output is always 60 + newline char (\n)
  4. Width of output is always 45
  5. Empty on right side of line must be filled with spaces
  6. Numbers are always odd (even numbers generate ugly ascii art)
  7. Each line must have newline char (\n) on the end
  8. Each line doesn't have additional blank chars on the end

Description

Task is simple - write the code to draw easter egg: {{ terminal(help_fancy_egg) }}

First argument is a domain small language (DSL) to draw pattern on the egg.

Format of argument is: 

command1,arg1:command2,arg1,arg2,…,argN:command3,arg1,arg2:…:commandN,arg1
Don't be afraid! List of commands is below on this page

To help you with a task I have made maskoff command to show you how draw patterns:
With mask: {{ terminal(help_basic_egg) }} Without mask: {{ terminal(help_maskoff) }}

Egg have always a constant shape {{ terminal(help_egg) }}

Commands

zigzag

zigzag:CHAR,SIZE {{ terminal(help_zigzag) }}

maze

maze:CHAR,VSIZE,HSIZE {{ terminal(help_maze) }}

cross

cross:CHAR,SIZE {{ terminal(help_cross) }}

hstrip

hstrip:CHAR,HSIZE,HSIZE {{ terminal(help_hstrip) }}

vstrip

vstrip:CHAR {{ terminal(help_vstrip) }}

empty

vstrip:HSIZE
just draw N empty (ekhm! filled with spaces!) lines

maskoff command is not required
